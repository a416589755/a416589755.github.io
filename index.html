hexo+github搭建个人博客

hexo是目前最主流的静态博客框架，该框架基于node，因此编译效率非常高，同事借助github可以实现陵城本搭建和维护，Let’s go!(0o0)

环境搭建

安装node环境

在node官网点击下载
对应操作系统的node安装包，安装过程比较简单，就
是next模式。安装完毕之后通过以下步骤检测是否安装成功:

  • win+r打开命令窗口
  • 输入cmd回车
  • 在cmd窗口输入node -v

如果提示”node不是外部或者内部命令时”则表示安装失败

安装git

git:版本控制器
进入git官网点击下载对应操作
系统的git安装包，next模式安装

vscode是一款强大的IDE

hexo+node+git搭建个人博客

一、环境/工具的安装

1.node

(一款牛叉的web服务器，目前很多大型项目都使用node搭建。)

  • 特点：解析效率高，没有兼容性。
  • 比较适io密集型的项目。
  • 版本：建议版本是6.10(比较稳定，兼容性好)
  • 如何安装？
  • node下载 下载相应操作的安装包
  • 安装过程下一步下一步
  • 如何测试？
  • WIN+R输入CMD回车
  • 在新打开的命令行输入node -v
  • 如果能显示node版本则安装成功，否则重新安装

    2.vscode(多语言开发工具)

    3.git(版本控制器)

二、安装hexo脚手架

1.WIN+R输入cmd回车

2.在CMD窗口输入npm install -g hexo-cli

  • npm(依赖包管理工具)
  • install(导入/安装)
  • -g(全局安装)

    3.初始化一个博客项目

  • hexo init 项目名称(英文)
  • init:初始化

    四、进入项目，添加依赖包

    1.进入到项目主目录文件夹

    2.鼠标右键选择git bash here

    3.使用npm install命令添加依赖包

  • 目的：将项目所需依赖包添加到本地

    五、启动服务器

    1.在blog博客目录右键git bash here

    2.输入hexo server

    3.在浏览器输入localhost：4000回车预览

    六、发表文章

  • hexo new 文章名

    七、博客预览

    1.Ctrl+s 保存

    2.hexo g 编译

    3.回到浏览器Ctrl+r 刷新
